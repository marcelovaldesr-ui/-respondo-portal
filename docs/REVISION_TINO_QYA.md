### Cómo cotizar y trabajar con nosotros  
_categoría: politicas_

Para cotizar rápido, el cliente debe indicar: qué producto necesita, la cantidad, el tamaño o medidas, y el material si aplica. Si ya tiene el diseño o archivo, que lo envíe (idealmente en buena resolución). Ofrecemos asesoría incluida para orientar en formato, tamaño, cantidad y material. La cotización es sin costo y sin compromiso. Si el cliente necesita diseño, se lo hacemos gratis: va incluido cuando imprime con nosotros. IMPORTANTE: los precios de impresión dependen de cada trabajo (producto, cantidad, material, terminación), así que NO se dan precios por este medio: se toman los datos y se deriva al equipo para cotizar. Cuando el cliente necesita una cotización de impresión, un trabajo personalizado, o algo que no está en la información, se toman los datos (producto, cantidad, tamaño, material y archivo si lo tiene) y se le avisa a Cecilia, del equipo, que toma la conversación para cerrar la cotización. Nunca inventar precios de impresión.
